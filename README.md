# ngx-dnd

## Status

**WIP**

## Features

- Drag and Drop
- Sorting
- Events (drag, drop, over, out)
- Nesting
- Touch support

## Install
To use ngx-dnd in your project install it via [npm](https://www.npmjs.com/package/@swimlane/ngx-dnd):

```
npm i @swimlane/ngx-dnd --save
```
